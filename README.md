# APG BOCHUM SECURE PRO

## 📱 Produktions-Datenerfassungssystem

Eine webbasierte Anwendung für die Erfassung von Produktionsdaten.

### ✨ Features

- 🔐 **Sicheres Login-System** mit persönlichen Passwörtern
- 👥 **Mitarbeiterverwaltung** nach Schichten (A, B, C)
- 🏭 **Anlagenübersicht** (PUR, Spritzguss, Compound)
- 📊 **Ausschuss- & Störungserfassung** mit vordefinierten Codes
- 📅 **Nachtschicht-kompatible Datumserfassung** (22:00 - 06:10)
- 👨‍💼 **Admin-Dashboard** mit Live-Berichten
- 📤 **Google Sheets Export**

### 🛠️ Technologien

- HTML5, CSS3, JavaScript
- Google Apps Script Backend
- Responsive Design
